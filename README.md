# Regional Pulse News

A news aggregator that translates regional headlines into English with brief summaries. Covers South America, Mexico, Central America, and Europe.

## Features
- Multi-region, multi-country news feeds
- Automatic translation to English
- Light/dark mode
- Date range filters: 24h, 3d, 7d, 30d
- Search and topic filtering
- Pre-enriched headline caching for fast loads
- PWA support

## Stack
- **Frontend:** Next.js on Vercel
- **Backend:** Python (FastAPI/uvicorn) on Render
